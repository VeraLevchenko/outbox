# Outbox Frontend

React приложение для просмотра файлов входящей и исходящей корреспонденции.

## Технологии

- **React 18** - UI библиотека
- **Vite** - быстрая сборка
- **Axios** - HTTP клиент для API запросов
- **Google Docs Viewer** - просмотр документов

## Установка и запуск

### 1. Установить зависимости

```bash
npm install
```

### 2. Запустить dev сервер

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000

### 3. Сборка для продакшена

```bash
npm run build
```

## Функциональность

### Экран "Входящие"

- Отображение списка входящих файлов из `/mnt/doc/Входящие/`
- Выделение главного документа
- Просмотр файлов через Google Docs Viewer
- Поддержка PDF, DOCX, XLSX

### Экран "Исходящие"

- Отображение главного DOCX файла (начинается с "исх_")
- Отображение приложений к письму
- Просмотр всех файлов через Google Docs Viewer

### Компоненты

- **App.jsx** - главный компонент с табами
- **IncomingFiles.jsx** - экран входящих файлов
- **OutgoingFiles.jsx** - экран исходящих файлов
- **FileViewer.jsx** - просмотр файлов через Google Viewer

### API интеграция

Приложение использует backend API:
- `GET /api/files/incoming/{card_id}` - получить входящие файлы
- `GET /api/files/outgoing/{card_id}` - получить исходящие файлы

Proxy настроен в `vite.config.js`, все запросы к `/api/*` проксируются на `http://localhost:8000`.

## Mock-данные

В режиме разработки backend использует mock-данные:
- 3 входящих файла для каждой карточки
- Исходящие DOCX файлы с приложениями

## Выбор карточки

В шапке приложения есть селектор для выбора карточки:
- 1001 - Письмо в Минфин о налоговых льготах
- 1002 - Договор на поставку оборудования
- 2001 - Отчет о проделанной работе

## Требования

- Node.js 18+
- NPM 9+
- Backend API запущен на http://localhost:8000
